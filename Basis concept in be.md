# 1️⃣ Programming Fundamentals (MUST be solid)

If this is weak → everything else collapses.

### ✅ You should be comfortable with:

- OOP principles
    
    - Encapsulation
        
    - Inheritance
        
    - Polymorphism
        
    - Abstraction
        
- Clean code basics
    
- Exception handling
    
- Collections (List, Map, Set…)
    
- Time & Space complexity (Big-O)
    
- Basic data structures:
    
    - Array
        
    - Linked List
        
    - Stack / Queue
        
    - HashMap
        
    - Tree (basic traversal)
        
- Recursion
    

If you cannot explain these clearly → fix this first.

# 2️⃣ Backend Core Concepts (Very Important)

This is where many students are weak.

## 🌐 HTTP & Web Basics

- What is HTTP?
    
- Difference between GET vs POST
    
- PUT vs PATCH
    
- Status codes (200, 201, 400, 401, 403, 404, 500)
    
- What is REST?
    
- What makes an API RESTful?
    
- What is stateless?
    

You should understand:

- Request → Server → Response
    
- Headers
    
- Body
    
- JSON
    

---

## 🗄️ Database Knowledge

### SQL (MUST know)

- SELECT, INSERT, UPDATE, DELETE
    
- JOIN (INNER, LEFT)
    
- GROUP BY
    
- Index
    
- Primary key vs Foreign key
    
- Transaction
    
- ACID
    

You should understand:

- What happens when many users write to DB?
    
- What is isolation level?
    
- What is normalization?
    

Bonus:

- Basic NoSQL concept (e.g. difference SQL vs MongoDB)
